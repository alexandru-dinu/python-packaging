# Python Packaging Reference

Simple, no-hassle reference for packaging a Python project using just `pyproject.toml`.

## Usage

```console
$ python3 -m venv ./.venv
$ source .venv/bin/activate
$ pip3 install -e .
$ pip3 install -e ".[dev]"
```
